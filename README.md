# Pachakari Express

A simple grocery order queue management system built using Flask and MySQL.

## Features

* Customer order submission
* Admin dashboard for pending orders
* Order completion workflow
* MySQL database integration
* Parameterized SQL queries
* Safe database connection handling using try-finally

## Tech Stack

* Python
* Flask
* MySQL
* HTML5
* CSS3
* Git

## Setup

1. Install Python and MySQL
2. Create database `grocery_db`
3. Run the provided SQL schema
4. Install dependencies:

pip install flask mysql-connector-python

5. Run:

python app.py

6. Open:

http://127.0.0.1:5000

Admin Dashboard:

http://127.0.0.1:5000/admin
